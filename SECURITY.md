# Security Policy

## Supported surface

This public repository mainly hosts documentation and issue workflows.
The production app runs at https://petmorph.app/.

## Reporting a vulnerability

Please do not post security issues publicly.

1. Use the support/contact channel on https://petmorph.app/
2. Include:
- A clear description of the issue
- Reproduction steps
- Impact assessment
- Any proof-of-concept details
3. Mark the report as security-related

We will review and triage reports as quickly as possible.

## Disclosure

After mitigation, we may publish a brief advisory when appropriate.
