# Public Roadmap

This roadmap reflects public-facing priorities and may evolve.

## Current focus

- Improve generation quality and consistency across styles
- Continue UX polish for upload-to-result flow
- Improve dashboard clarity for credit usage and generation history
- Strengthen multilingual and SEO coverage for key product pages

## Next milestones

1. Generation quality iteration
- Better style fidelity and prompt tuning
- Reduced failed generations and clearer retry paths

2. Account and billing UX
- Clearer credit package communication
- Better in-app purchase and usage feedback

3. Reliability and operations
- More automated checks for deployment and content quality
- Better monitoring and faster issue triage

4. Public documentation quality
- Keep changelog current
- Expand troubleshooting and FAQ references
